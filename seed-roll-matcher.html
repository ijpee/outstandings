Standalone Tool

Seed Roll ID Matcher

Paste a seed roll and a one-column list of student names. The matcher will try to resolve each name to the correct student ID without touching the existing KAMAR, Hero, or Edge flows.

Expected Input

Seed Roll

Accepts Kindo JSON or tab/comma-separated seed roll text.

Name List

One name per line. Supports formats like LAST NAME, First Name, uppercase surname tails, extra spaces, and nickname hints in parentheses.

Duplicates

Duplicate pasted lines are collapsed in the main output, with the occurrence count preserved.

Review Queue

Only students with multiple plausible matches appear here for manual review.

Unmatched

No reliable match was found. Suggested candidates are shown when available.

Summary

Student ID Output

One row for every pasted line. Unresolved rows show MISSING.
